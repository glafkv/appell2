Purpose:
	The main purpose of this project is to get familiar with using fork, exec, and shared memory. The command line needs to be able to take in 5 command line options. First, -h for a help page. -n for the max total of child processes, with the default option of 4. -s will be how many children should be allowed to exit, with the default option of 2.
